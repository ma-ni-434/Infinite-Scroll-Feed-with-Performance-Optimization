# Infinite Scroll Feed with Performance Optimization (React Native)

A high-performance React Native mobile app that displays thousands of feed items using paginated API calls with smooth infinite scrolling.

## Features
- FlatList virtualization for large datasets
- Infinite scroll with prefetching
- Pull-to-refresh
- Loading spinner + shimmer placeholders
- Error handling & retry
- Node.js + Express + MongoDB backend
- Optimized rendering for smooth performance

## Tech Stack
React Native, Android Studio, FlatList, Axios, Node.js, Express, MongoDB

## Run App
npm start
npx react-native run-android

## APK

- Virtualized FlatList
- Lazy lo
